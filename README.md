# THE KEEPER 
*Empathy without discernment is dangerous—those who exploit it won’t even have to lie to get what they want.*

## About the project
The Keeper is a 3D low-poly task simulator game with retro, PSX inspired graphics. You play as Lola Angelina, a sweet and God-fearing old lady who 
does her best to keep things nice and tidy. When faced with unbearable odds, would Lola Angelina break or would she decide to give in to dark depths of her morality?

## Getting Started

### Requirements
Godot 4.6 stable

### How to run
1. Clone the repo into your computer
2. Import to Godot
3. Click run
